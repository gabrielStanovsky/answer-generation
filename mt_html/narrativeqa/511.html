Task Instruction:

In this task, you are given a passage, a question, a correct answer, and a predicted answer. The task is to score how closely a predicted answer captures the same information as a correct answer using the passage, question, and human commonsense knowledge.

Scoring is on a scale from 1 to 5 .
A score of 1 means the predicted answer captures none of the information of the correct answer, and thus is a completely wrong answer.
A score of 5 means the predicted answer captures the same information as the correct answer, and thus is a completely right answer.

Each HIT will contain multiple questions and predictions for the same context so it needs to only be read once per HIT.


Note: A predicted answer may be a correct answer, but to be given a score of 5, it must be equivalent to the correct answer. An example is provided below of this case.

Note: The score you give should not depend on the capitalization of the correct and predicted answer.

Note: A helpful tip for searching for the correct answer in the passage is using Control+F to search for it.

Examples:

Correct Answer: grandaughter
